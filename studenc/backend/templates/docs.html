POST /api/postcompany/

Creates company if it doesnt exist and returns id



            request body
            name:companyname

                "id": 15,
                "name": "companyname"
            }


POST /api/postjob/

Creates job if it does not yet exist



            request body
            title:jobtitle
            location:joblocation
            description:jobdescription
            spots:jobspots
            code:jobcode
            neto:jobneto
            bruto:jobbruto
            phone:jobphone
            email:jobemail
            contact:jobcontact
            company:jobcompany
            date:jobdate
        


            job posted successfully | job already exists
